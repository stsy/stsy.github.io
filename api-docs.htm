API Documentation

This API provides the current week number and date/time based on the user's local system clock.

Endpoint

GET https://yourusername.github.io/week-number.html?format=json

Response

The response is a JSON object with the following structure:

{
    "weekNumber": <Number>,
    "dateTime": "<String>"
}

Example Response

{
    "weekNumber": 42,
    "dateTime": "10/17/2023, 3:24:15 PM"
}

Usage

To get the current week number in JSON format, make a GET request to the endpoint above.

Notes

  • The week number is calculated based on the ISO 8601 standard.
  • The date and time are based on the user's local system clock.
  • As this is a static site, the Content-Type header is not set to application/json.
